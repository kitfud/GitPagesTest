Basic Web Development

HTML

HTML stands for the HyperText Markup Language. It is made up of a set of markup tags that describe the contents of the document.

CSS

CSS stands for Cascading Style Sheets. It is used along with a markup language like HTML to specify how the document looks. You can use CSS to style the different elements of a document by specifying different element properties such as color or font.

JavaScript

JavaScript is a programing language that is used to add functionality to our webpages. We can use it to interact with elements like making something happen when a button is clicked. Or we can use it to HTML content, attributes, or style.
